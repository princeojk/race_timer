🏁 Race Timer

A modern race timing application built to accurately record participant times, manage race results, and provide an intuitive interface for organisers.

Features
- ⏱️ Start and stop race timers
- 👥 Register and manage participants
- 🏆 Automatically calculate finishing positions
- 📊 View race results in real time
- 📱 Responsive interface for desktop and mobile
- ⚡ Fast and lightweight application
- 🔄 Easy to extend with additional race formats

Tech Stack
- Frontend: Javascript
- Backend: Node.js 
- Database: sqlite 
- Styling: CSS 
- Version Control: Git & GitHub
